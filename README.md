# Boeing — Base de Conhecimento de Aeronaves Comerciais

Boeing — Base de Conhecimento é uma aplicação web voltada para estudantes da área de aviação, entusiastas da história aeronáutica e qualquer pessoa interessada em explorar de forma simples e rápida os modelos e variantes de aeronaves comerciais produzidas pela Boeing ao longo de sua história.

A aplicação reúne fichas técnicas resumidas de modelos icônicos — desde variantes clássicas como o 707 e o 727, até famílias modernas como 737 MAX, 777X e 787 Dreamliner — permitindo consulta organizada, filtrada e acessível a todos.

# Objetivo do Projeto

O projeto tem como propósito:
- Disponibilizar uma base educacional sobre aeronaves comerciais da Boeing.
- Oferecer uma ferramenta de consulta simples, visual e responsiva.
- Permitir a pesquisa por modelo, variante, tags e ano de criação.
- Facilitar a navegação para quem estuda aviação, engenharia aeronáutica, ou apenas aprecia a dinastia Boeing.

# Tecnologias Utilizadas

- HTML5 (estrutura)
- CSS3 com foco em responsividade e UI limpa
- JavaScript Vanilla (controle de busca, filtro e renderização dinâmica)
- JSON como fonte de dados
- Nenhum framework necessário — aplicação leve e rápida

# Como Utilizar

1. Utilize a barra de pesquisa para localizar modelos específicos.
2. Use filtros para limitar resultados por status (legado / em produção).
3. Ajuste a ordenação conforme sua necessidade.
3. Clique em Saiba mais para acessar fontes externas confiáveis.
